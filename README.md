# Sum of Even Numbers in C

This program finds the **sum of all even numbers from 1 up to N**.
The user enters a number `N`, and the program adds all even numbers (like 2, 4, 6 â€¦) until `N`, then shows the result.

### How to Run

1. Save the code as `even_sum.c`
2. Compile with:

   ```bash
   gcc even_sum.c -o even_sum
   ```
3. Run with:

   ```bash
   ./even_sum
   ```
